Open Source

Below is a list of some of the open-source software projects I've created and contributed to. Visit my GitHub profile to see more.


Slapper.AutoMapper

Slapper.AutoMapper is a .NET library that can convert dynamic data into static types and populate complex nested child objects.

Contributions: Project creator, primary contributor, and former maintainer. The library has been downloaded on NuGet over 2.3 million times, has multiple contributors, and has spun off into its own organization at which point I handed over ownership to the community. This project is still in active use by thousands of organizations worldwide.

Nano

Nano is a .NET cross-platform micro web framework for rapidly building web-based HTTP services and websites. It can run self-hosted or within an ASP.NET website.

Contributions: Project creator, primary contributor, and former maintainer. The framework was used to create over 60 enterprise web applications and had over 15 contributors. This project is now archived and was superseded by ASP.NET Core.

Sequelocity.NET

Sequelocity is a .NET micro-ORM data access library providing lightweight ADO.NET wrapper, object mapper, and helper functions with support for MS SQL Server, MySql, SQLite, and PostgreSQL.

Contributions: Project creator, primary contributor, and former maintainer. The library was used in over 100 enterprise applications and I created extensive documentation and contributed over 500 unit tests to ensure high code quality. This project is now archived and was superseded by Dapper.
